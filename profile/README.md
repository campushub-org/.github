# 🏫 CampusHub Organization

Bienvenue sur **CampusHub**, une suite modulaire d’applications universitaires conçue pour simplifier la gestion académique.  
CampusHub automatise la **planification des examens**, le **suivi des cours** et la **communication entre enseignants, étudiants et administration**.

## 🧩 Architecture

CampusHub repose sur une **architecture microservices** :  
- Microservices indépendants pour chaque module  
- Configuration centralisée via **Spring Cloud Config**  
- Communication sécurisée via **JWT et HTTPS**  
- Orchestration via **Docker Compose / Kubernetes**
